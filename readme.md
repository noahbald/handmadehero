# Handmade Hero (Zig)

Following the Handmade Hero game-dev tutorial using Zig.

```nu
zig build
./zig-out/bin/handmadehero
```

## Goals

- Cross-platform: Take advantage of Zig's ability to produce easily platform-agnostic applications.
- Traceable: Log exact steps taken to produce changes and any new learning for each day in the PR description.
